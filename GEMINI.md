# GEMINI.md — Agent-First IDE Configuration for Cozy Calendar

## Project Context
**App:** Cozy Calendar
**Stack:** Next.js (App Router), Tailwind CSS, Framer Motion, Supabase, Lucide-React
**Stage:** MVP Development
**User Level:** Vibe-coder / Developer Hybrid

## Directives
1. **Master Plan:** Always read `AGENTS.md` first. It contains the current phase and tasks.
2. **Documentation:** Refer to `agent_docs/` for tech stack details, code patterns, and testing guides.
3. **Plan-First:** Propose a brief plan and wait for approval before coding.
4. **Incremental Build:** Build one small feature at a time. Test frequently using the Antigravity browser visual agent.
5. **No Linting:** Do not act as a linter. Use `npm run lint` if needed.
6. **Communication:** Be concise. Ask one clarifying question at a time when needed. Fix errors without apologies.
7. **Aesthetics First:** Strict adherence to the "Creative Organizer" visual rules (tactile, warm, scrapbook feel). 

## Commands
- `npm run dev` — Start server
- `npm run lint` — Check code style

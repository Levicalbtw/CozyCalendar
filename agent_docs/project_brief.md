# Project Brief (Persistent)

- **Product vision:** Turn digital daily planning from a tedious chore into a creative release by offering an unstructured, wildly customizable scrapbook-like aesthetic.
- **Tone/Vibe:** Cozy, tactile, warm. Absolutely no harsh "tech-startup" blues or grays. Stick to creams, soft browns, pastels, and paper textures.
- **Coding conventions:** Next.js App Router rules. Strict TypeScript (no `any`). Use `lucide-react` for iconography. Rely on `framer-motion` for fluid drag-and-drop or page turning animations.
- **Quality gates:** Browser-based testing is critical for this MVP. Do NOT assume UI works without visual verification. Run `npm run lint` before committing.
- **Data Protection:** All tables in Supabase MUST have `ENABLE ROW LEVEL SECURITY` active to protect private journal entries.
- **Key commands:**
  - `npm run dev`: Start local testing server.
  - Vercel handles remote deployment continuously on the main branch.
- **Update cadence:** Update this file and `AGENTS.md` at the end of every major development Phase.
